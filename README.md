# AURA — Autopoietic Unified Resilience Architecture

> *Not a simulation of life, but a simulation of the functions that make life capable of persisting, learning, and cooperating.*

AURA is a seven-level framework for cognitive augmentation systems. Each level simulates a distinct biological function that living systems have developed to maintain **cognitive continuity despite entropy**.

## The Seven Levels

| Level | Module | Biological Function | Operative Principle |
|-------|--------|--------------------|--------------------|
| 1 | [WAAA](https://github.com/MarBeo-cyber/waaa) | Informational persistence | Maintain internal coherence by filtering noise |
| 2 | [MAAA](https://github.com/MarBeo-cyber/MAAA) | Situational stabilisation under stress | **Reduce, don't explain** |
| 3 | [PAAA](https://github.com/MarBeo-cyber/PAAA) | Neuro-functional continuity | Deviation from self, not from population norm |
| 4 | [SAAA](https://github.com/MarBeo-cyber/SAAA) | Knowledge consolidation | Associative learning with selective reinforcement |
| 5 | [XAAA](https://github.com/MarBeo-cyber/XAAA) | Mediated experience and operational wisdom | The experiential cycle converts knowledge into judgement |
| 6 | [TAAA](https://github.com/MarBeo-cyber/TAAA) | Inter-schema cognitive translation | Active interference before it becomes declared conflict |
| 7 | [NOAH](https://github.com/MarBeo-cyber/NOAH) | Collective cognitive cooperation | Aggregated schema geometry with preserved individual signature |

## The Three Categorical Jumps

1. **WAAA → MAAA**: from monitoring to situational action
2. **SAAA → XAAA**: from knowledge to wisdom
3. **TAAA → NOAH**: from individual to network

## Five Original Contributions

1. **Schema Memory M0/M1/M2** — memory of the relational structure of an individual's cognitive schemas
2. **LLM latent space as cognitive Rosetta Stone** — measuring inter-schema distance without exposing individual contents
3. **SARS score** — first computational metric for detecting active interference vs. ignorance
4. **'Reduce, don't explain'** — output architecture under cognitive stress (MAAA)
5. **Wisdom Trace** — computational object formalising the difference between declarative knowledge and operational wisdom

## The Noosphere with Consent

NOAH is the first computational analogue of Teilhard de Chardin's noosphere — with explicit consent in place of spontaneous emergence, and preservation of individual diversity in place of convergence toward an Omega Point.
